두레이 아카이빙 체계의 접근성 한계를 개선하고, 다년부터 축적된 매체·기자 데이터를 실질적인 업무 자산으로 전환하기 위한 웹·앱 기반 리포트 프로그램입니다.

01

추진 배경

현행 미팅로그 관리 방식의 구조적 한계와 데이터 활용 공백에 대한 진단

기존 시스템의 접근성 및 편의성 한계

  • 두레이 '홍보실-매체 관리' 프로젝트 내 게시글 형태로 아카이빙되어, 일원화된 검색·조회 불가
  • 특정 매체·기자의 과거 이력 확인 시 게시글 개별 열람이 필요해 미팅 사전 준비 시간 과다 소요
  • 모바일 환경에서의 실시간 작성 및 확인에 대한 사용자 편의성 저하 지속 제기
  • 이동 중·미팅 현장 등 비고정 업무 환경에서의 즉시 기록 곤란

대규모 축적 데이터의 활용성 부재

  • 다년간 누적된 2천 건 이상의 미팅로그와 1천여 명 규모의 기자 데이터, 단순 텍스트 적재 수준에 머물러 있음
  • 매체별·기자별 통계, 인물 프로필, 트렌드 인사이트 등으로 전환할 데이터 자산화 체계 부재
  • 인물 정보의 개인 기억·수기 메모 의존도가 높아, 담당자 교체·인수인계 시 관계 히스토리 유실 위험 상존
02

개요

미팅 데이터의 신속한 파악과 접근성 극대화를 위한 통합 리포트 프로그램의 목적 및 방향성

목적

  • 미팅 데이터의 신속한 파악과 미팅로그 접근성 극대화를 위한 웹·앱 기반 통합 매체 관리 리포트 프로그램 개발
접근성

접근성 혁신

언제 어디서나 미팅로그 확인 및 기록이 가능하도록 모바일·웹 환경 최적화, 앱 설치 없이 홈 화면 아이콘 추가를 통한 네이티브 앱 수준의 이용 환경 구현

시각화

데이터 시각화

축적된 매체·기자 데이터의 대시보드화를 통해 기간별 미팅 추이, 매체 카테고리 분포, 매체·기자별 접촉 순위 직관적 파악 가능

자산화

인물 정보 자산화

미팅로그 본문에 산재된 기자 개개인의 인적사항(연차·출입처·학력·이전 소속 매체 등) 자동 추출 및 정리를 통한 즉시 조회 체계 구축

효율화

업무 효율화

임원 보고용 리포트 자동 생성, 데이터 일괄 다운로드 등 반복 업무 자동화를 통한 미팅 데이터 탐색·공유 시간 단축

보안

보안 강화

접근 코드 기반 사용자 인증 및 다운로드 파일 암호화를 통한 민감 취재원 정보의 안전한 관리 체계 확보

링크·앱 원클릭 진입

별도 설치 파일 없이 전달받은 링크 하나로 즉시 접속 가능하며, 모바일 홈 화면에 아이콘 형태로 추가하면 네이티브 앱과 동일한 방식으로 실행됩니다. PWA(Progressive Web App) 구조로 구현되어 있어 브라우저 주소창이나 별도 로딩 없이 아이콘 터치 한 번으로 바로 진입됩니다.

* 상기 UI는 데스크탑 포털과 완벽히 연동되는 모바일 최적화 웹 앱(PWA) 구성 예시 화면입니다.
